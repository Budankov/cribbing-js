Шпаргалка по JS за конспектом GoIT

Модуль 1 - Заняття 1 - Змінні та типи

Number - цілі числа і числа з плаваючою комою (крапкою). Прикладconst age = 20; const points = 15.8;

String - рядки, послідовність з нуля або більше символів. Прикладconst username = "Mango"; const description = "JavaSript для початківців";

Boolean - логічний тип даних, прапорці стану. Всього два значення: true і false. Прикладconst isLoggedIn = true; const canMerge = false; const hasChildren = true; const isModalOpen = false;

null - особливе значення, яке по суті означає ніщо. Прикладlet selectedProduct = null;

undefined - ще одне спеціальне значення за замовчуванням, коли змінна оголошується, але не ініціалізується, її значення не визначено, їй присвоюється undefined. Прикладlet username; console.log(username); // undefined

typeof - використовується для отримання типу значення змінної. Повертає на місце свого виклику тип значення змінної. Прикладlet username; console.log(typeof username); // "undefined" let inputValue = null; console.log(typeof inputValue); // "object" const quantity = 17; console.log(typeof quantity); // "number" const message = "JavaScript is awesome!"; console.log(typeof message); // "string" const isSidebarOpen = false; console.log(typeof isSidebarOpen); // "boolean"

NaN - значенням, що становить не число (Not-A-Number). Прикладconsole.log(Number(valueB)); // NaN

Number(val) - приведення до числа, передаючи їй у val те, що потрібно привести до числа. Прикладconst valueA = "5"; console.log(Number(valueA)); // 5 console.log(typeof Number(valueA)); // "number" const valueB = "random string"; console.log(Number(valueB)); // NaN console.log(typeof Number(valueB)); // "number"

.isNaN(val) - перевіряє, чи вказане значення є NaN, true - якщо значення val - NaN, false - якщо значення val - не NaN. Прикладconst validNumber = Number("51"); // 51 console.log(Number.isNaN(validNumber)); // false const invalidNumber = Number("qweqwe"); // NaN console.log(Number.isNaN(invalidNumber)); // true

.parseInt() - парсить з рядка ціле число. Прикладconsole.log(Number.parseInt("5px")); // 5 console.log(Number.parseInt("12qwe74")); // 12 console.log(Number.parseInt("12.46qwe79")); // 12 console.log(Number.parseInt("qweqwe")); // NaN

.parseFloat() - парсить з рядка дробове число. Прикладconsole.log(Number.parseFloat("5px")); // 5 console.log(Number.parseFloat("12qwe74")); // 12 console.log(Number.parseFloat("12.46qwe79")); // 12.46 console.log(Number.parseFloat("qweqwe")); // NaN

toFixed() - спосіб додати, а результат скоротити до певного знаку після коми. Прикладconsole.log(0.17 + 0.24); // 0.41000000000000003 console.log((0.17 + 0.24).toFixed(2)); // 0.41

Math - один із вбудованих класів, який надає набір методів для роботи з числами. Прикладтекст

Math.floor(num) - повертає найбільше ціле число, менше, або яке дорівнює зазначеному числу. Прикладconsole.log(Math.floor(1.7)); // 1

Math.ceil(num) - повертає найменше ціле число, більше, або яке дорівнює зазначеному числу. Прикладconsole.log(Math.ceil(1.2)); // 2

Math.round(num) - повертає значення числа, округленого до найближчого цілого. Прикладconsole.log(Math.round(1.2)); // 1 console.log(Math.round(1.5)); // 2

Math.max(num1, num2, ...) - повертає найбільше ціле число з набору Прикладconsole.log(Math.max(20, 10, 50, 40)); // 50

Math.min(num1, num2, ...) - повертає найменше ціле число з набору. Прикладconsole.log(Math.min(20, 10, 50, 40)); // 10

Math.pow(base, exponent) - піднесення до степеня. Прикладconsole.log(Math.pow(2, 4)); // 16

Math.random() - повертає псевдовипадкове число в діапазоні [0, 1). Прикладconsole.log(Math.random()); // випадкове число між 0 і 1 console.log(Math.random() * (10 - 1) + 1); // псевдовипадкове число від 1 до 10

${вираз} - шаблонний рядок. Приклад// Використовуючи змінні, необхідно скласти рядок з підставленими значеннями const guestName = "Манго"; const roomNumber = 207; const greeting = "Welcome " + guestName + ", your room number is " + roomNumber + "!"; console.log(greeting); // "Welcome Mango, your room number is 207!"

.length - відображає довжину рядка, тобто кількість його символів. Прикладconst message = "Welcome to Bahamas!"; console.log(message.length); // 19 console.log("There is nothing impossible to him who will try".length); // 47

.toLowerCase() і .toUpperCase() - повертають новий рядок у відповідному регістрі, не змінюючи оригінальний рядок. Прикладconst message = "Welcome to Bahamas!"; console.log(message.toLowerCase()); // "welcome to bahamas!" console.log(message.toUpperCase()); // "WELCOME TO BAHAMAS!" console.log(message); // "Welcome to Bahamas!"

.indexOf() - повертає позицію (індекс), на якій знаходиться перший збіг підрядка або -1, якщо нічого не знайдено. Прикладconst message = "Welcome to Bahamas!"; console.log(message.indexOf("to")); // 8 console.log(message.indexOf("hello")); // -1

.includes() - перевіряє, чи міститься підрядок в рядку, повертає буль - true, якщо міститься, і false - в іншому випадку. Прикладconst productName = "Ремонтний дроїд"; console.log(productName.includes("н")); // true console.log(productName.includes("Н")); // false console.log(productName.includes("дроїд")); // true console.log(productName.includes("Дроїд")); // false console.log(productName.includes("Ремонтний")); // true console.log(productName.includes("ремонтний")); // false

.endsWith() - дозволяє визначити, чи завершується рядок символами (підрядком), зазначеними в дужках, повертаючи true або false. Прикладconst jsFileName = "script.js"; console.log(jsFileName.endsWith(".js")); // true const cssFileName = "styles.css"; console.log(cssFileName.endsWith(".js")); // false

.replace() і .replaceAll() - повертає новий рядок, в якому перше (replace) або усі збіги (replaceAll) підрядка замінені на вказане значення. Прикладconst jsFileName = "script.js"; const minifiedJsFileName = jsFileName.replace(".js", ".min.js"); console.log(minifiedJsFileName); // "script.min.js" const cssFileNames = "styles.css, about.css, portfolio.css"; const minifiedCssFileNames = cssFileNames.replaceAll(".css", ".min.css"); console.log(minifiedCssFileNames); // "styles.min.css, about.min.css, portfolio.min.css"

.slice() - метод рядків slice (startIndex, endIndex) використовується для створення копії. Він робить копію від startIndex і до, але не включно endIndex і повертає новий рядок, не змінюючи оригінал. Прикладconst productName = "Repair droid"; console.log(productName.slice(0, 4)); // "Repa" console.log(productName.slice(3, 9)); // "air dr" console.log(productName.slice(0, productName.length)); // "Repair droid" console.log(productName.slice(7, productName.length)); // "droid"

&& - логічне «І», приводить всі операнди до буля і повертає значення одного з них. Прикладconsole.log(1 && 5); // true && true -> 5 console.log(5 && 1); // true && true -> 1 console.log(0 && 2); // false && true -> 0 console.log(2 && 0); // true && false -> 0 console.log("" && "Mango"); // false && true -> "" console.log("Mango" && ""); // true && false -> "" console.log("Mango" && "Poly"); // true && true -> "Poly" console.log("Poly" && "Mango"); // true && true -> "Mango"

|| - логічне «АБО», приводить всі операнди до буля і повертає значення одного з них. Прикладconsole.log(true || false); // true console.log(false || true); // true console.log(true || true); // true console.log(3 || false); // 3 console.log(false || 3); // 3 console.log(3 || true); // 3 console.log(true || 3); // true

!вираз - логічне «НЕ», оператор ! приводить операнд до буля, якщо необхідно, а потім робить інверсію - змінює його на протилежний true -> false або false -> true. Прикладconsole.log(!true); // false console.log(!false); // true console.log(!3); // !3 -> !true -> false console.log(!"Mango"); // !"Mango" -> !true -> false console.log(!0); // !0 -> !false -> true console.log(!""); // !"" -> !false -> true const isOnline = true; const isNotOnline = !isOnline; // !isOnline -> !true -> false

Модуль 1 - Заняття 2 - Розгалуження та цикли

if - «ЯКЩО», виконує інструкцію, якщо зазначена умова виконується (істинно). Якщо умова не виконується (хибно), може бути виконана інша інструкція. Прикладтекст

if...else - «ЯКЩО»...ІНШЕ», розширює синтаксис if таким чином, що якщо умова приводиться до false, виконається код у фігурних дужках після оператора else. Прикладтекст

else...if - «ІНШЕ»...«ЯКЩО», дозволяє додати після else ще один оператор if з умовою. Прикладтекст

<умова> ? <вираз_якщо_умова_правдива> : <вираз_якщо_умова_хибна> - ьернарний оператор використовується у якості синтаксично коротшої заміни інструкції if...else. Прикладтекст

switch - інструкція розгалуження, «плоскіший» синтаксис if...else. Прикладтекст

while - цикл з передумовою — цикл, який виконується, доки правдива якась умова, зазначена до його початку. Прикладтекст

do...while - цикл з постумовою - цикл, в якому умова перевіряється після виконання тіла циклу. Прикладтекст

for - цикл з лічильником - цикл, в якому певна змінна змінює своє значення із заданого початкового до кінцевого значення з деяким кроком. Прикладтекст

break - повністю припиняє виконання циклу і передає управління на рядок після його тіла. Прикладтекст

continue - виконання тіла припиняється і управління передається до наступної ітерації. Прикладтекст

Модуль 2 - Заняття 3 - Масиви

[] - літерал масиву, кожен елемент масиву розділяється комою. Масив - структура даних для зберігання і маніпулювання колекцією індексованих значень. Прикладтекст

.split() - перетворює рядок в масив, «розбиваючи» його роздільником. Якщо роздільник - це порожній рядок, то створиться масив окремих символів. Роздільником може бути один або декілька символів. Прикладтекст

.join() - об'єднує елементи масиву у рядок. У рядку елементи будуть розділені символом або групою символів. Тобто ця операція протилежна методу рядків .split(). Прикладтекст

.indexOf() - повертає перший індекс, в якому елемент зі значенням value був знайдений в масиві, або число -1, якщо такий елемент відсутній. Прикладтекст

.includes() - перевіряє, чи містить масив елемент зі значенням value і повертає true або false відповідно. Прикладтекст

.push() - додає один або декілька елементів наприкінці масиву, без необхідності зазначати індекси елементів, що додаються. Прикладтекст

.pop() - видаляє останній елемент з кінця масиву і повертає видалений елемент. Якщо масив порожній, метод повертає undefined. Прикладтекст

.slice(begin, end) - повертає новий масив, що містить копію частини вихідного масиву, не змінюючи його. Копія створюється з begin і до, але не включно, end - індекси елементів вихідного масиву. Прикладтекст

.splice(position, num, sum) - видаляє, додає і замінює елементи у довільному місці масиву. position - вказує на позицію (індекс) першого елемента для видалення, num - визначає кількість елементів, що видаляються, sum - вказує початкову позицію в масиві, куди будуть вставлені нові елементи. Прикладтекст

.concat() - об'єднує два або більше масивів в один. Він не змінює масив, на якому викликається, а повертає новий. Прикладтекст

Модуль 2 - Заняття 4 - Функції

function назва() {} - це підпрограма, незалежна частина коду, призначена для багаторазового виконання конкретної задачі з різними початковими значеннями. назва() - виклики функції назва. Прикладтекст

return - використовується для передачі значення з тіла функції у зовнішній код, він відразу ж виходить з функції (припиняє її виконання), і повертає вказане значення у те місце коду, де була викликана функція. Прикладтекст

arguments - можна отримати доступ до списку всіх аргументів, яка доступна тільки всередині функції і зберігає всі аргументи у якості псевдомасиву. Прикладтекст

Array.from() - метод який створить масив із псевдомасиву. Array.from(arguments). Прикладтекст

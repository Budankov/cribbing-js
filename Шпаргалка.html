Модуль 1 - Заняття 1 - Змінні та типи Number - цілі числа і числа з плаваючою комою (крапкою). String - рядки, послідовність з нуля або більше символів. Boolean - логічний тип даних, прапорці стану. Всього два значення: true і false. null - особливе значення, яке по суті означає ніщо. undefined - ще одне спеціальне значення за замовчуванням, коли змінна оголошується, але не ініціалізується, її значення не визначено, їй присвоюється undefined. typeof - використовується для отримання типу значення змінної. Повертає на місце свого виклику тип значення змінної. NaN - значенням, що становить не число (Not-A-Number). Number(val) - приведення до числа, передаючи їй у val те, що потрібно привести до числа. Number.isNaN(val) - перевіряє, чи вказане значення є NaN, true - якщо значення val - NaN, false - якщо значення val - не NaN. Number.parseInt() - парсить з рядка ціле число. Number.parseFloat() - парсить з рядка дробове число toFixed() - спосіб додати, а результат скоротити до певного знаку після коми. Math - один із вбудованих класів, який надає набір методів для роботи з числами. Math.floor(num) - повертає найбільше ціле число, менше, або яке дорівнює зазначеному числу. Math.ceil(num) - повертає найменше ціле число, більше, або яке дорівнює зазначеному числу. Math.round(num) - повертає значення числа, округленого до найближчого цілого. Math.max(num1, num2, ...) - повертає найбільше ціле число з набору Math.min(num1, num2, ...) - повертає найменше ціле число з набору. Math.pow(base, exponent) - піднесення до степеня. Math.random() - повертає псевдовипадкове число в діапазоні [0, 1). ${вираз} - шаблонний рядок. .length - відображає довжину рядка, тобто кількість його символів. .toLowerCase() і .toUpperCase() - повертають новий рядок у відповідному регістрі, не змінюючи оригінальний рядок. .indexOf() - повертає позицію (індекс), на якій знаходиться перший збіг підрядка або -1, якщо нічого не знайдено. .includes() - перевіряє, чи міститься підрядок в рядку, повертає буль - true, якщо міститься, і false - в іншому випадку. .endsWith() - дозволяє визначити, чи завершується рядок символами (підрядком), зазначеними в дужках, повертаючи true або false. .replace() і .replaceAll() - повертає новий рядок, в якому перше (replace) або усі збіги (replaceAll) підрядка замінені на вказане значення. .slice() - метод рядків slice (startIndex, endIndex) використовується для створення копії. Він робить копію від startIndex і до, але не включно endIndex і повертає новий рядок, не змінюючи оригінал. && - логічне «І», приводить всі операнди до буля і повертає значення одного з них. || - логічне «АБО», приводить всі операнди до буля і повертає значення одного з них. !вираз - логічне «НЕ», оператор ! приводить операнд до буля, якщо необхідно, а потім робить інверсію - змінює його на протилежний true -> false або false -> true. Модуль 1 - Заняття 2 - Розгалуження та цикли if - «ЯКЩО», виконує інструкцію, якщо зазначена умова виконується (істинно). Якщо умова не виконується (хибно), може бути виконана інша інструкція. if...else - «ЯКЩО»...ІНШЕ», розширює синтаксис if таким чином, що якщо умова приводиться до false, виконається код у фігурних дужках після оператора else. else...if - «ІНШЕ»...«ЯКЩО», дозволяє додати після else ще один оператор if з умовою. <умова> ? <вираз_якщо_умова_правдива> : <вираз_якщо_умова_хибна> - ьернарний оператор використовується у якості синтаксично коротшої заміни інструкції if...else. switch - інструкція розгалуження, «плоскіший» синтаксис if...else. while - цикл з передумовою — цикл, який виконується, доки правдива якась умова, зазначена до його початку. do...while - цикл з постумовою - цикл, в якому умова перевіряється після виконання тіла циклу. for - цикл з лічильником - цикл, в якому певна змінна змінює своє значення із заданого початкового до кінцевого значення з деяким кроком. break - повністю припиняє виконання циклу і передає управління на рядок після його тіла. continue - виконання тіла припиняється і управління передається до наступної ітерації. Модуль 2 - Заняття 3 - Масиви [] - літерал масиву, кожен елемент масиву розділяється комою. Масив - структура даних для зберігання і маніпулювання колекцією індексованих значень. .split() - перетворює рядок в масив, «розбиваючи» його роздільником. Якщо роздільник - це порожній рядок, то створиться масив окремих символів. Роздільником може бути один або декілька символів. .join() - об'єднує елементи масиву у рядок. У рядку елементи будуть розділені символом або групою символів. Тобто ця операція протилежна методу рядків .split(). .indexOf() - повертає перший індекс, в якому елемент зі значенням value був знайдений в масиві, або число -1, якщо такий елемент відсутній. .includes() - перевіряє, чи містить масив елемент зі значенням value і повертає true або false відповідно. .push() - додає один або декілька елементів наприкінці масиву, без необхідності зазначати індекси елементів, що додаються. .pop() - видаляє останній елемент з кінця масиву і повертає видалений елемент. Якщо масив порожній, метод повертає undefined. .slice(begin, end) - повертає новий масив, що містить копію частини вихідного масиву, не змінюючи його. Копія створюється з begin і до, але не включно, end - індекси елементів вихідного масиву. .splice(position, num, sum) - видаляє, додає і замінює елементи у довільному місці масиву. position - вказує на позицію (індекс) першого елемента для видалення, num - визначає кількість елементів, що видаляються, sum - вказує початкову позицію в масиві, куди будуть вставлені нові елементи. .concat() - об'єднує два або більше масивів в один. Він не змінює масив, на якому викликається, а повертає новий. Модуль 2 - Заняття 4 - Функції function назва() {} - це підпрограма, незалежна частина коду, призначена для багаторазового виконання конкретної задачі з різними початковими значеннями. назва() - виклики функції назва. return - використовується для передачі значення з тіла функції у зовнішній код, він відразу ж виходить з функції (припиняє її виконання), і повертає вказане значення у те місце коду, де була викликана функція. arguments - можна отримати доступ до списку всіх аргументів, яка доступна тільки всередині функції і зберігає всі аргументи у якості псевдомасиву. Array.from() - метод який створить масив із псевдомасиву. Array.from(arguments).
